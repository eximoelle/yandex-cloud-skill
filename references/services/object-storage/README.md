# Object Storage Module

## Scope

Модуль для Object Storage: bucket/prefix/object операции, S3 Select и доступы.

## In This Module

- `configuration.md` -> baseline конфигурация bucket и auth
- `api.md` -> API/SDK/S3-совместимые точки входа
- `patterns.md` -> upload/list/audit паттерны
- `gotchas.md` -> частые проблемы доступа и формата

## Reading Order

- Интеграция storage -> `configuration.md` -> `api.md` -> `patterns.md`
- Ошибки доступа -> `gotchas.md`
