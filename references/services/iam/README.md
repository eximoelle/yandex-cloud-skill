# IAM Service Module

## Scope

Модуль для задач IAM: сервисные аккаунты, роли, IAM токены, политика доступа.

## In This Module

- `configuration.md` -> scope, роли, базовые настройки
- `api.md` -> canonical API/SDK точки входа
- `patterns.md` -> рабочие паттерны для automation
- `gotchas.md` -> частые ошибки и фиксы

## Reading Order

- Новая интеграция -> `configuration.md` -> `patterns.md` -> `api.md`
- Ошибка доступа -> `gotchas.md` -> `configuration.md`
