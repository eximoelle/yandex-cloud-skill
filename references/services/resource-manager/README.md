# Resource Manager Module

## Scope

Модуль для работы с организацией, облаками и каталогами (`organization/cloud/folder`).

## In This Module

- `configuration.md` -> как выбирать и фиксировать scope
- `api.md` -> canonical операции и источники
- `patterns.md` -> инвентаризация и управление иерархией
- `gotchas.md` -> ошибки контекста и доступа

## Reading Order

- Inventory/scope задачи -> `configuration.md` -> `patterns.md`
- Ошибки `NotFound`/`PermissionDenied` -> `gotchas.md`
