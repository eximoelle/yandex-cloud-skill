# VPC Module

## Scope

Модуль для сети: `network`, `subnet`, маршрутизация, security groups.

## In This Module

- `configuration.md` -> базовая топология сети
- `api.md` -> canonical API/SDK источники
- `patterns.md` -> типовые сетевые паттерны
- `gotchas.md` -> сетевые ошибки и диагностика

## Reading Order

- Новая сеть -> `configuration.md` -> `patterns.md`
- Сетевые проблемы -> `gotchas.md`
