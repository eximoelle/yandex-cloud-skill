# Compute Cloud Module

## Scope

Модуль для задач по VM/Compute Cloud: создание, листинг, lifecycle и troubleshooting.

## In This Module

- `configuration.md` -> минимальные параметры VM
- `api.md` -> API/SDK точки входа
- `patterns.md` -> шаблоны создания и безопасных изменений
- `gotchas.md` -> частые ошибки provisioning

## Reading Order

- Новая VM -> `configuration.md` -> `patterns.md` -> `api.md`
- Ошибка создания VM -> `gotchas.md`
