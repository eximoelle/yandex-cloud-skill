## Adaptation Checklist

1. Приведи `folder_id`, `region`, `zone` к контексту задачи.
2. Проверь IAM prerequisites.
3. Для задач по базовой инфраструктуре сначала сверяй решение с `infrastructure/#tools`.
4. Убери параметры, не нужные для минимального рабочего сценария.
5. Добавь команды `terraform fmt`, `terraform validate`, `terraform plan`.
